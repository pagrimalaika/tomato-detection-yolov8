# 🍅 Tomato Maturity and Defect Detection using YOLOv8

## 📌 Deskripsi
Project ini bertujuan untuk mendeteksi tingkat kematangan dan cacat pada buah tomat pascapanen menggunakan metode YOLOv8 (You Only Look Once). Sistem ini dikembangkan untuk membantu proses sortasi tomat agar lebih cepat, akurat, dan efisien.

---

## 🎯 Tujuan
- Mengidentifikasi tingkat kematangan buah tomat
- Mendeteksi cacat pada tomat
- Meningkatkan efisiensi proses sortir pascapanen

---

## 🧠 Teknologi yang Digunakan
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Google Colab / Jupyter Notebook
